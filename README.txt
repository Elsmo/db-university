База данных University отражает структуру университета. 
Это может быть полезно как для студентов, так и для абитуриентов.
По таблицам нашей БД можно посмотреть проходные баллы и количество мест
на интересующее направление. Студенты могут посмотреть расписание факультативов, 
преподаватели могут легко найти старост групп. 

Таблица faculties показывает факультеты.

В таблице specialties можно посмотреть сколько предусмотрено мест на каждой специальности

Таблица students скорее подойдет для преподавателей, потому что в ней отражены ФИО студентов, информация об их группе,
курсе и среднем балле диплома. 

По таблице passing_scores абитуриенты могут понять, каковы их шансы на поступление в университет.

Таблицы electives и electives_timetable предназначены для студентов. По этим таблицам они могут выбрать себе факультативы 
или уточнить время начала занятий.

С помощью таблицы elective_students преподаватели и другие студенты могут посмотреть, кто уже записался на 
факультативы. 

Таблица group_leaders поможет преподавателям найти старост групп для того чтобы передать им важную информацию.

Таблицы enrollees и enrollees_documents помогут абитуриентам оценить конкуренцию на места.
